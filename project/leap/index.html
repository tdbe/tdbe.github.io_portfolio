"Leap" Puzzle Platformer
ITU 3Mo 4Man Project
Dec 2012

|A 4-person design and development project during the first semester of my MSc.

This project focused on the design process, mechanics, and evaluation, more than production value.


| The concept revolves around swapping between alternate dimensions.
In a mirage-like state, the character can teleport between the real world and a hallucinated plane. As the game progresses the two worlds (planes) intertwine.


| My constraints / mechanics proposal:

| The mechanics: Switch planes at will. Fire projectile to teleport target object or enemy to opposite plane. Have the ability to offset opposite plane's position.
Entities are either physically bound to both planes, have different versions of themselves on each plane, or only inhabit one plane.
